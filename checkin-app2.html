🔗 GitHub Sync einrichten

Die App speichert den Check-In Status direkt in Ihrem GitHub Repository. Beide Computer sehen die Daten in Echtzeit — kein SharePoint, kein Azure nötig.

1
Gehen Sie zu github.com → Profilbild → Settings → Developer settings → Personal access tokens → Tokens (classic)
2
„Generate new token (classic)" → Haken bei repo → Token kopieren
3
Token unten eintragen und auf „Verbinden" klicken — wird nur in Ihrem Browser gespeichert
Der Token wird nur lokal in Ihrem Browser gespeichert und nie übertragen.
